Creating WAPT packages
======================

.. attention::

  Installing *tis-pyscripter* will install PyScripter :term:`IDE`.
  :program:`PyScripter` is required for packaging WAPT packages.

  Please first visit the documentation on :ref:`setting up the environment
  for developing WAPT packages <envdev_setup>`.

To create and customize WAPT packages, follow that documentation
and you will quickly become a master of WAPT.
